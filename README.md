# Exercícios de Certificados Digitais e Servidor HTTPS

## Descrição do Projeto

Este repositório contém os artefatos e documentação referentes aos exercícios práticos sobre certificados digitais, infraestrutura de chaves públicas (PKI) e configuração de servidor HTTPS. Os exercícios foram realizados em ambiente Linux (Debian) utilizando ferramentas como OpenSSL, SCP e Python.
